截止 2026 年 1 月 4 日，我的官服绝区零站点已满员，不可再申请！

关于域名更换的公告

本站从建立之初，一直在使用 dora-honor.github.io 作为主要访问域名。

考虑到默认 username.github.io 域名对于 SEO 优化不太友好，同时想更有个性，因此已购买并绑定了 dorahonor.me 作为新的主要访问域名。

与此同时，考虑到国内访问 GitHub 和 GitHub Pages 会比较慢，本站也使用了 Cloudflare CDN 加速访问，提升访问速度。

目前，dorahonor.me 已经正式启用，所有访问 dora-honor.github.io 的请求都会自动重定向到 dorahonor.me。

感谢大家一直以来的支持与关注！如果有任何问题或建议，欢迎通过博客的留言功能与我联系。

- Dora Honor 于 2025 年 12 月 21 日
